SonarLint for Visual Studio version 2.0.0

SonarLint for Visual Studio version 2.0 is a new major release of SonarLint with support for connecting SonarLint to a SonarQube instance. SonarLint relies on the new SonarAnalyzer C# and VB.NET Nuget packages, which were previously shipped as a single SonarLint Nuget package. To avoid any further confusion from now on we are refering to the Nuget packages by the name SonarAnalyzer, and to the Visual Studio extension by SonarLint.
