SonarAnalyzer 1.11.0

  • Release date: April 6, 2016
  • Release notes for C#
  • Download C# Nuget
  • Download VB.NET Nuget

SonarAnalyzer 1.11.0 adds 6 new rules to the product. Furthermore, 13 rules have been improved to not report false positives or just simply to have better issue messages. Finally, two bugs have also been fixed on rules Rule S125 and Rule S2952. Here are the new rules since version 1.10.0:

  • Return values should not be ignored when function calls don't have any side effects - Rule S2201
  • Methods should not return values that are never used - Rule S3241
  • Classes should not have only "private" constructors - Rule S3453
  • Exceptions should not be explicitly rethrown - Rule S3445
  • Empty "case" clauses that fall through to the "default" should be omitted - Rule S3458
  • Namespaces should not be empty - Rule S3261
